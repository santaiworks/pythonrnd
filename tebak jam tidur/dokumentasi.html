Dokumentasi: Tebak Jam Tidur via Python

Script tebak_tidur.py ini berfungsi untuk menebak pola tidur pengguna berdasarkan histori penggunaan laptop (kapan mati/sleep dan kapan nyala kembali).

Cara Kerja Singkat:
Script ini tidak "memasang alat penyadap". Sebaliknya, ia hanya membaca Windows Event Logs yang sudah ada di sistem laptop Anda, khususnya event yang mencatat kapan laptop tertidur (Sleep/Shutdown) dan terbangun (Wake/Boot).

Mengapa Menggunakan Library Ini?

Berikut adalah alasan pemilihan library (modul) yang digunakan dalam script ini:

1. subprocess

Fungsi: Menjalankan perintah terminal/PowerShell dari dalam Python.

Kenapa pakai ini?

  • Tanpa Install Tambahan: Kita tidak perlu menginstall library eksternal yang ribet seperti pywin32. modul subprocess adalah bawaan Python.
  • PowerShell Lebih Kuat: Mengakses Event Log Windows via PowerShell (Get-WinEvent) sangat powerful dan fleksibel tanpa perlu low-level coding C++ atau API Windows yang rumit.
2. json

Fungsi: Mengolah data teks menjadi format object/dictionary Python.

Kenapa pakai ini?

  • Output Bersih: PowerShell diperintahkan untuk mengeluarkan data dalam format JSON (ConvertTo-Json). Python sangat mudah membaca format ini dibanding harus memotong-motong (parsing) teks biasa secara manual.
3. datetime & timedelta

Fungsi: Mengelola waktu dan tanggal.

Kenapa pakai ini?

  • Hitung Durasi: Kita perlu menghitung selisih waktu (durasi) antara laptop mati dan nyala. Library ini membuatnya sangat mudah (tinggal waktu_b - waktu_a).
  • Format Tanggal: Membantu mengubah timestamp komputer yang aneh menjadi format yang bisa dibaca manusia (contoh: "Senin, 03 Feb 23:00").

Logika Code (Algorithm)

Proses penebakan dilakukan dengan langkah-langkah berikut:

  1. Ambil Data (Extract): Script menjalankan PowerShell untuk mencari Event ID tertentu:
    • 42: Kernel-Power (Sleep)
    • 1074/6006: Shutdown (Mati)
    • 1/6005: Wake/Boot (Nyala)
  2. Urutkan (Sort): Data diurutkan berdasarkan waktu kejadian.
  3. Cari Celah (Analyze Gaps): Script mencari jarak waktu (gap) antar event. 
    Jarak = Waktu Event Sekarang - Waktu Event Sebelumnya
  4. Filter (Heuristics): Tidak semua laptop mati berarti tidur. Script menggunakan aturan:
    • Durasi mati harus lebih dari 4 jam.
    • Waktu mulai mati harus di jam "wajar" tidur (Malam hari) ATAU waktu nyala di pagi hari.
  5. Hitung Rata-rata: Menghitung rata-rata jam tidur dan jam bangun dari data yang valid.

Contoh Output

HARI       | TANGGAL      | TIDUR    -> BANGUN   | DURASI    
-----------------------------------------------------------------
Senin      | 03 Feb       | 23:15    -> 07:30    | 8.2 jam   
Selasa     | 04 Feb       | 00:45    -> 08:15    | 7.5 jam
